# Phone Store + Portfolio (Flask)
Single-file Flask backend with templates for a simple business site that supports:
- Phones with price + image
- Graphic designs (free/paid) with preview and optional downloadable file
- Videos (uploaded and played)
- Admin panel to upload/manage content
- About page with downloadable CV
